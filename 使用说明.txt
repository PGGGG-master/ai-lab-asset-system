AI 实验室资产管理系统 — 业务源码包（不含环境依赖）

【本包包含】
- backend/app/          后端 Python 源码
- backend/requirements.txt
- backend/static/       内置 Web 页面（可选）
- backend/uploads/      演示资产文件
- frontend/src/         前端 Vue 源码
- frontend/package.json、vite.config.js、index.html
- README.md、start.bat

【本包不包含】
- backend/venv/         Python 虚拟环境
- frontend/node_modules/  Node 依赖
- frontend/dist/        前端构建产物
- ai_lab_assets.db      数据库（首次启动自动生成）

【在新电脑运行】

1. 后端
   cd backend
   python -m venv venv
   venv\Scripts\python.exe -m pip install -r requirements.txt
   venv\Scripts\python.exe -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000

2. 前端（可选，需 Node.js）
   cd frontend
   npm install
   npm run dev

3. 或双击 start.bat（需先完成后端 venv 安装）

详细说明见 README.md
